When scrolling Reels in Instagram DMs, each video opens in a popup and you have to manually close it before opening the next one, which means too many repetitive clicks.

This extension automatically closes the current reel and opens the next one for you, so you can keep scrolling without all the extra taps.

## Features

*   **Automatic:** Automatically detect reels shared in DMs.

*   **Smooth:** Scroll to next reel with a smooth UI.

*   **Simple:** User-friendly controls for navigation.


## How to Run

1. **Go to the **[Releases](../../releases)** section on the right.**


2. Download the .crx(for chrome) and.zip(for all others)

3. Open your browser and visit chrome://extensions/.

4. Enable Developer mode.

5. Drag and drop the downloaded file into the Extensions page.

6. Click Add Extension to confirm.
